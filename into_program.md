## Linguagem Lua

![lua](https://upload.wikimedia.org/wikipedia/commons/c/cf/Lua-Logo.svg)

Criada em 1993 na PUC-Rio.

### Tipos de dados
- nil
- boolean
- number
- string
- function
- table

O tipo nil possui nil como seu ´único valor, de modo que nenhum outro valor é igual a nil. Em geral, usamos esse tipo para indicar a ausˆencia de um valor. 

O tipo boolean representa os valores booleanos true e false.

Ao contrário da maioria das linguagens, Lua possui somente um tipo numérico básico, number, que é ponto flutuante por padrão.

O tipo string representa cadeias de caracteres, as quais podem ser delimitadas por aspas duplas ou simples. 

Function é o tipo que representa funções.

O tipo userdata é usado para armazenar dados C em variáveis Lua.

Valores do tipo thread representam fluxos de execução independentes.

Por fim, o tipo table representa uma tabela Lua. Tabelas são o único mecanismo de estruturação de dados de Lua. Com tabelas podemos representar arrays, tabelas de síımbolos, conjuntos, grafos, registros, etc. Podemos usar qualquer valor para indexar uma tabela, exceto nil.
